## **Dishcovery — Discover Dishes by Ingredients:**
Dishcovery is a smart recipe generator built with React JS.
Enter the ingredients you have, and the app instantly suggests delicious recipes you can make — helping you discover dishes from what’s already in your kitchen!

## **Features:**
1.Search by ingredients — Type three or more ingredients to get matching recipes. \
2.Recipe discovery — View detailed recipes, ingredients, and instructions. \
3.Clean UI — Simple, modern, and responsive interface. \
4.Real-time results — Uses API calls to fetch live recipe data. \
5.Dynamic filtering — Finds the best matches based on user input.

## **Tech Stack:**
- Category    ->	Tools 
- Frontend    ->	React JS (Hooks, Components, State Management) 
- Styling     ->	CSS / Tailwind  
- Build Tool  ->	Vite / Create React App 
- Deployment  ->	Netlify
